## 📖 読み方ガイド

### 基本的な使い方

1. **目的の機能を見つける**
   - 上記のカテゴリから実装したい機能を探す
   - 関連する機能も確認（例：アイテム→インベントリ→装備）

2. **ドキュメントを読む**
   - 各ドキュメントには以下が含まれます：
     - 機能の概要
     - データ構造
     - 主要な処理フロー
     - 使用例
     - 関連機能へのリンク

3. **実装する**
   - サンプルコードを参考に実装
   - 必要に応じて[カスタマイズガイド](../guides/)を参照

### 機能間の関連性

多くの機能は互いに関連しています：

```
戦闘 ← 装備・スキル・状態異常・敵AI
  ↓
報酬 ← キャラクター成長・アイテム
  ↓
インベントリ ← アイテム・装備・クラフト・強化
```

### 実装の順序（推奨）

初めて実装する場合の推奨順序：

1. **基礎**: キャラクター成長、アイテム、インベントリ
2. **戦闘**: 戦闘、装備、スキル習得
3. **発展**: 状態異常、敵AI、報酬
4. **応用**: クラフト、強化、ジョブ変更
5. **システム**: セーブロード、データ永続化

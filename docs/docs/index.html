ドキュメント

rpg-coreは、JRPG（日本のロールプレイングゲーム）スタイルのターン制RPGを作成するための TypeScriptライブラリです。このドキュメントでは、各機能の詳細な使い方を説明します。

各機能のドキュメント

rpg-coreは、以下の主要な機能を提供しています。各機能の詳細については、それぞれのページをご覧ください。

アーキテクチャ

rpg-coreは、以下のレイヤー構造を持っています：

レイヤー構造

  1. UI層 - 見た目と入力の処理
  2. Headless UI層 - 選択・確認・キャンセルの操作フロー
  3. JRPG Services層 - 操作フローの管理
  4. Core Engine層 - 数値計算とルール判定
  5. Game State層 - 永続可能な状態

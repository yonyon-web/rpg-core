ドキュメント

GEasy Kitは、JRPG（日本のロールプレイングゲーム）スタイルのターン制RPGを作成するための TypeScriptライブラリです。手軽で柔軟な設計により、ゲーム開発をもっとシンプルにします。

アーキテクチャ

GEasy Kitは、以下のレイヤー構造を持っています：

レイヤー構造

  1. UI層 - 見た目と入力の処理
  2. Headless UI層 - 選択・確認・キャンセルの操作フロー
  3. JRPG Services層 - 操作フローの管理
  4. Core Engine層 - 数値計算とルール判定
  5. Game State層 - 永続可能な状態

次のステップ

  • クイックスタート - すぐに始める
  • カスタマイズガイド - ゲーム固有の実装方法

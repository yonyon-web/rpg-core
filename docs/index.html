GEasy Kitの特徴

✨

手軽に始められる

シンプルなAPIで、すぐにゲーム開発を始められます。複雑な設定は不要です。

🔧

柔軟にカスタマイズ

ゲーム固有のルールに対応できる拡張性を備えた設計

🎯

純粋な計算とルール

UIや操作フローに依存しない、数値計算とゲームルールの判定に特化

🧪

テスト容易

モックや単体テストが容易に書ける設計で、品質の高い開発が可能

📘

TypeScript完全対応

型定義付きで開発体験が向上し、エディタの支援を最大限活用

クイックスタート

インストール

npm install geasy-kit

基本的な使い方

import { GEasyKit } from 'geasy-kit';

// ライブラリ全体を初期化
const game = new GEasyKit({
  config: customGameConfig,  // 省略可能
  useEventBus: true          // 省略可能
});

// サービスに簡単にアクセス
const battleService = game.services.battle;
const itemService = game.services.item;

// コントローラーも簡単に作成
const battleController = game.controllers.battle();

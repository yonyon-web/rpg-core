📈

キャラクター成長

🎓

スキル習得

👥

パーティ編成

メンバー追加・削除、並び替え、パーティ管理

🎒

アイテム・装備

インベントリ管理、装備変更、アイテム使用

🛠️

クラフト・強化

アイテム合成、装備強化、素材管理

🤖

敵AI

敵の行動決定、グループ管理、戦術パターン

💫

状態異常・バフ

🎭

ジョブ変更

ショップ

JRPG設計思想

日本のRPGスタイルに最適化された、戦闘・成長・管理システム

クイックスタート

インストール

npm install GEasy-Kit

基本的な使い方

import { RPGCore } from 'GEasy-Kit';

// ライブラリ全体を初期化
const rpg = new RPGCore({
  config: customGameConfig,  // 省略可能
  useEventBus: true          // 省略可能
});

// サービスに簡単にアクセス
const battleService = rpg.services.battle;
const itemService = rpg.services.item;

// コントローラーも簡単に作成
const battleController = rpg.controllers.battle();

次のステップ

  • ドキュメントを読む - ライブラリの詳細な使い方を学ぶ
  • サンプルを試す - 実際の動作を確認する
  • APIリファレンス - 利用可能なクラスとメソッドを確認

アーキテクチャ

UI（見た目・入力）
↓
Headless UI（選択・確認・キャンセル）
↓
JRPG Services（操作フロー）
↓
Core Engine（数値・ルール）
↓
Game State（永続可能な状態）

レイヤー化された設計により、各層が明確な責任を持ち、 テストやカスタマイズが容易になっています。

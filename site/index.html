rpg-core

ターン制RPGゲームを作る仕組みを提供するTypeScriptライブラリ

はじめる GitHub

特徴

🎯

純粋な計算とルール

UIや操作フローに依存しない、数値計算とゲームルールの判定に特化

🔄

状態を持たない設計

Game Stateから必要なデータを受け取り、計算結果を返す決定論的な動作

🧪

テスト容易

モックや単体テストが容易に書ける設計で、品質の高い開発が可能

🎨

カスタマイズ可能

ゲーム固有のルールに対応できる拡張性を備えた設計

📘

TypeScript完全対応

型定義付きで開発体験が向上し、エディタの支援を最大限活用

🏗️

JRPG設計思想

日本のRPGスタイルに最適化された、戦闘・成長・管理システム

クイックスタート

インストール

npm install rpg-core

基本的な使い方

import { RPGCore } from 'rpg-core';

// ライブラリ全体を初期化
const rpg = new RPGCore({
  config: customGameConfig,  // 省略可能
  useEventBus: true          // 省略可能
});

// サービスに簡単にアクセス
const battleService = rpg.services.battle;
const itemService = rpg.services.item;

// コントローラーも簡単に作成
const battleController = rpg.controllers.battle();

アーキテクチャ

UI（見た目・入力）
↓
Headless UI（選択・確認・キャンセル）
↓
JRPG Services（操作フロー）
↓
Core Engine（数値・ルール）
↓
Game State（永続可能な状態）

レイヤー化された設計により、各層が明確な責任を持ち、 テストやカスタマイズが容易になっています。

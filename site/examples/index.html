サンプルコード・使用例

rpg-coreの実践的な使用例を紹介します。各サンプルは実際のゲーム開発で使えるコードです。

基本セットアップ

初級

RPGCoreの基本的な初期化とサービスへのアクセス方法

import { RPGCore } from 'rpg-core';

// 基本的な初期化
const rpg = new RPGCore();

// サービスにアクセス
const battleService = rpg.services.battle;
const itemService = rpg.services.item;
const partyService = rpg.services.party;

console.log('RPG Core initialized successfully!');

戦闘システムの使用

中級

戦闘の開始、コマンド実行、結果の取得

import { RPGCore } from 'rpg-core';

const rpg = new RPGCore();
const battleService = rpg.services.battle;

// 戦闘を開始
const battleState = battleService.startBattle({
  playerParty: [hero1, hero2],
  enemyParty: [enemy1, enemy2, enemy3]
});

// コマンドを選択
const command = {
  type: 'attack',
  actorId: hero1.id,
  targetId: enemy1.id
};

// コマンドを実行
const result = battleService.executeCommand(
  battleState, 
  command
);

console.log('Battle result:', result);

装備システム

中級

装備の変更と能力値の計算

import { RPGCore } from 'rpg-core';

const rpg = new RPGCore();
const equipmentService = rpg.services.equipment;

// 装備を変更
const character = getCharacter('hero1');
const weapon = getItem('legendary_sword');

const result = equipmentService.equipItem(
  character,
  weapon,
  'weapon'
);

if (result.success) {
  console.log('Equipped successfully!');
  console.log('Attack increased by:', result.statChanges.attack);
  
  // 前の装備を返却
  if (result.previousItem) {
    inventoryService.addItem(result.previousItem);
  }
}

イベント処理

中級

EventBusを使用したイベント駆動の処理

import { RPGCore } from 'rpg-core';

const rpg = new RPGCore({ useEventBus: true });
const eventBus = rpg.eventBus;

// イベントリスナーを登録
eventBus.on('battle:start', (data) => {
  console.log('Battle started!', data);
});

eventBus.on('character:levelUp', (data) => {
  console.log('Level up!', data);
  showLevelUpAnimation(data.character);
});

eventBus.on('item:used', (data) => {
  console.log('Item used:', data.itemId);
  showItemEffect(data);
});

// イベントは各サービスが自動的に発行
battleService.startBattle(/* ... */);

セーブ・ロード

中級

ゲーム状態の保存と復元

import { RPGCore } from 'rpg-core';

const rpg = new RPGCore();
const saveService = rpg.services.save;

// ゲーム状態を保存
const gameState = {
  party: getCurrentParty(),
  inventory: getInventory(),
  progress: getGameProgress(),
  timestamp: Date.now()
};

await saveService.save('slot1', gameState);

// ゲーム状態を読み込み
const loadedState = await saveService.load('slot1');
if (loadedState) {
  restoreGameState(loadedState);
  console.log('Game loaded!');
}

カスタムダメージ計算

中級

独自のダメージ計算式を実装

import { RPGCore, DamageCalculator } from 'rpg-core';

class CustomDamageCalculator extends DamageCalculator {
  calculatePhysicalDamage(attacker, defender, skill) {
    // 独自の計算式
    const baseDamage = attacker.attack * 2;
    const defense = defender.defense;
    const levelDiff = attacker.level - defender.level;
    
    let damage = baseDamage - defense + levelDiff * 5;
    
    // クリティカルヒット判定
    if (Math.random() < 0.1) {
      damage *= 2.5;
    }
    
    return Math.max(1, damage);
  }
}

// カスタム計算機を使用
const rpg = new RPGCore({
  damageCalculator: new CustomDamageCalculator()
});

パーティ編成

初級

パーティメンバーの追加・削除・並び替え

import { RPGCore } from 'rpg-core';

const rpg = new RPGCore();
const partyService = rpg.services.party;

// パーティを作成
const party = partyService.createParty({
  maxSize: 4,
  members: [hero1, hero2]
});

// メンバーを追加
partyService.addMember(party, hero3);

// メンバーを削除
partyService.removeMember(party, hero2.id);

// 並び替え
partyService.reorderMembers(party, [
  hero1.id, 
  hero3.id
]);

console.log('Party:', party.members);

スキル習得

中級

キャラクターのスキル習得と管理

import { RPGCore } from 'rpg-core';

const rpg = new RPGCore();
const skillService = rpg.services.skill;

// 習得可能なスキルを確認
const availableSkills = skillService.getLearnableSkills(
  character
);

// スキルを習得
const skill = availableSkills[0];
const result = skillService.learnSkill(
  character, 
  skill.id
);

if (result.success) {
  console.log('Learned new skill:', skill.name);
} else {
  console.log('Cannot learn:', result.reason);
}

状態異常管理

中級

バフ・デバフ・状態異常の適用と管理

import { RPGCore } from 'rpg-core';

const rpg = new RPGCore();
const statusService = rpg.services.status;

// 状態異常を付与
statusService.applyStatus(character, {
  type: 'poison',
  duration: 3,
  power: 10
});

// バフを付与
statusService.applyStatus(character, {
  type: 'attackUp',
  duration: 5,
  power: 50  // 50%攻撃力上昇
});

// ターン経過
statusService.processTurn(character);

// 状態を確認
const statuses = statusService.getActiveStatuses(character);
console.log('Active statuses:', statuses);
